# Snake

A game of snake written in C++ using ncurses.

## Requirements

* gcc
* ncurses

## Installation

	make
	./snake
	
## Controls
	
Turn left:

	a or leftarrow
	
Turn right:

	d or rightarrow
	
Quit:

	q or ESC
