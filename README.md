# Snake

A game of snake written in C++ using ncurses.

<img src="media/snake.png" alt="Snake" height="80%" width="80%"/>

## Requirements

* gcc
* ncurses

## Installation

	make
	./snake
	
## Controls
	
Turn left:

	a or leftarrow
	
Turn right:

	d or rightarrow
	
Quit:

	q or ESC
